# Web Development Skills

## Stack preferencial
- Frontend: React com TypeScript, Tailwind CSS para estilização
- Use sempre componentes funcionais com hooks
- Prefira estados imutáveis e gerenciamento com Zustand ou Context API

## Convenções
- Nomeie componentes em PascalCase
- Nomeie funções e variáveis em camelCase
- Use arrow functions para componentes e handlers
- Separe lógica de negócio em hooks customizados (use*)

## Boas práticas
- Sempre valide inputs do usuário no cliente e no servidor
- Use variáveis de ambiente para configurações sensíveis (.env)
- Implemente error boundaries em componentes críticos
- Escreva código legível; prefira clareza à brevidade
- Use async/await em vez de .then()/.catch()
