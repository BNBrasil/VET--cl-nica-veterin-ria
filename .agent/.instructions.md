## 🆘 RESOLUÇÃO DE PROBLEMAS COMUNS

### "Cannot find module"
- Verificar caminho relativo correto
- Limpar `node_modules` e `npm install`
- Compilar TypeScript: `npm run build`

### "JWT Token inválido"
- Verificar `JWT_SECRET` no `.env`
- Validar expiração do token
- Verificar cookie httpOnly

### "CORS error"
- Validar `FRONTEND_URL` no backend
- Verificar `VITE_API_URL` no frontend
- Credentials: true em ambos os lados

### "Database connection failed"
- Verificar `DATABASE_URL`
- Validar credenciais e host
- Executar: `npx prisma db push`
